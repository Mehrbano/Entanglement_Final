# Entanglement_Final

/*******************************************************************
 *   Devloped by Mehrbano Khattak //view credicts 
 *   for final project of MA Computational Arts, Goldsmiths College, University of London
 **  NOTE
 *   This program is developed in Processing 2.2.1
